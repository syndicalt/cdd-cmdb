## Regenerate

```bash
git clone https://github.com/syndicalt/cdd-cmdb.git && cd cdd-cmdb
pip install -e ".[generator]"
export ANTHROPIC_API_KEY=sk-ant-...
python -m generator --profile minimal --badge
```
